Mouse Coordinates

  • Screen : Relative to the screen
    • mouseEvent.screenX
    • mouseEvent.screenY
  • Client : Relative to the viewport
    • mouseEvent.clientX
    • mouseEvent.clientY
  • Page : Relative to the page
    • mouseEvent.pageX
    • mouseEvent.pageY
  • Offset : Relative to the element
    • mouseEvent.offsetX
    • mouseEvent.offsetY

    Element Size

  • Offset : The size of the element - This includes border
    • element.offsetWidth
    • element.offsetHeight
  • Client : The size of the visible content - This excludes scroll bar
    • element.clientWidth
    • element.clientHeight
  • Scroll : The size of the whole content
    • element.scrollWidth
    • element.scrollHeight
  • Rendered : The exact rendered size of the element
    • element.getBoundingClientRect().width
    • element.getBoundingClientRect().height

    Window Size

  • Screen : The size of the screen
    • window.screen.width
    • window.screen.height
  • Window : The size of the window
    • window.outerWidth
    • window.outerHeight
  • Viewport : The size of the viewport - This includes the scroll bar
    • window.innerWidth
    • window.innerHeight
  • Document : The size of the document - This excludes the scroll bar
    • document.documentElement.clientWidth
    • document.documentElement.clientHeight
  • Page : The size of the content
    • document.documentElement.scrollWidth
    • document.documentElement.scrollHeight
